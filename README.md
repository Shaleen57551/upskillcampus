# ServiceHub - Multi-Client Website Offering Client Services

Repository for the UCT Full Stack Internship.

## Technology Stack
- **Backend:** Java 17, Spring Boot 3, Spring MVC, Spring Data JPA, Spring Security, JWT, Maven
- **Frontend:** HTML5, CSS3, Bootstrap 5, Vanilla JS
- **Database:** MySQL

## Modules to be implemented
- User Authentication (JWT)
- Merchant & Customer Dashboards
- Service Listings & Categories
- Booking & Payment Engine
- Ratings & Reviews
- Admin Console

## Instructions to run
1. Ensure MySQL is running on port 3306 with credentials root/root (or update `application.properties`).
2. Start the Spring Boot backend via Maven:
   ```bash
   cd backend
   mvn spring-boot:run
   ```
3. Open `frontend/index.html` in your browser.
